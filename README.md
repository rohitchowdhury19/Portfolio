## 🚀 Overview

A modern, high-performance, and visually stunning professional portfolio built with the latest web technologies. This project showcases my skills, projects, and professional journey with a focus on **Rich Aesthetics** and **Dynamic Interactions**.

> [!TIP]
> This portfolio features a unique "Text Scramble" effect, smooth scroll animations with Framer Motion, and a custom splash screen for a premium first impression.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Social Icons](https://jaketrent.github.io/react-social-icons/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/)
- **Email Service**: [Resend](https://resend.com/) / [@emailjs/browser](https://www.emailjs.com/)

---

## ✨ Key Features

- **Dynamic Hero Section**: Interactive intro with a text scrambling role switcher.
- **Premium Aesthetics**: Dark-themed UI with glassmorphism and subtle gradients.
- **Micro-Animations**: Smooth transitions and hover effects using Framer Motion.
- **Social Integration**: One-click access to LinkedIn, GitHub, Instagram, and X.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Contact Form**: Seamless communication via integrated email services.

## 🛠️ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/rohitchowdhury19/Portfolio.git
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
